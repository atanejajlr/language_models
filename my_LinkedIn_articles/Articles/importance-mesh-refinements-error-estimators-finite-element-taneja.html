The importance of mesh refinements and error estimators in a finite element analysis

Created on 2016-08-19 07:05

Published on 2016-08-19 07:13

A classical example to highlight the importance of mesh refinements in a finite element analysis is the Sleipper offshore platform disaster in Norway.

 

https://www.ima.umn.edu/~arnold/disasters/sleipner.html

 

It was the coarseness of the mesh that led to the underestimation of shear stresses and hence the disaster.

 

This could have been avoided had someone done the error estimates (such as error estimate based on strain energy, the Zienkiewicz and Zhu error estimator) and consequent adaptive mesh refinement process.

 

The above (importance of mesh refinements) is irrespective of the analysis type: linear or nonlinear.

This is because in structures exhibiting significant geometrical non linear effects, the deformation can be so large that the mesh used to model the materials may become highly distorted, and the analysis cannot go any further without using some special techniques.

 

The adaptive re-meshing (pointed out in the video above) is a useful feature to overcome these difficulties.

 

An extra step in such types of adaptive re-meshing (after non-linear iterations) is that one would need to interpolate the necessary data from the old mesh to the new mesh before the analysis for the subsequent non-linear iteration begins.
