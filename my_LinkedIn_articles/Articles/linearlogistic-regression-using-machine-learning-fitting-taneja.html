The equation for a straight line is usually written y=mx+b; where m is the slope and b is the y-intercept. This can be generalized to multiple feature dimensions as y=θ^Tx + b where θ and x are vectors. Here the θ elements are referred to as weights and b as bias.

You can think of bias as the “intrinsic” way that the system behaves even when all the features are zero. Large weight values are associated with overfitting and large values of bias with underfitting. In weeks 1 and 2 Prof Ng (course on courseera) did a mathematical “trick” to aid vectorization.

A dummy x0 element, always equal to 1, was added to x vector and a corresponding θ0. This removed the b term from the formulas. The new weights are often called bias terms or bias weights and the dummy elements bias units. This allows the hypothesis for any number of features to be computed with a single matrix multiplication.
