Example of industrial applications of Sentiment Analysis

There can be several industrial examples wherein a sentiment analysis through Machine Learning algorithms involving Natural Language Processing(NLP) can prove effective. Some of these being;

1) You're in the Software Sales Division of a company that develops, markets and sells a leading Engineering software. Recently you have invited customer feedback of several functionalities in the software. You've sold the software to several customers over the years and many of them have provided feedback following some major updates to the software. You have to classify the feedback in some form - AI based algorithms based on NLP could prove a good and robust AI based solution to automatically/ intelligently process customer feedback

2) You're in the HR division of a company and there have been some major re-organization in the company (well, this is quite common in the current scenario!) and different employees have reacted differently. You need a sentimental analysis system to classify employee feedback into: positive, negative and more criteria.

3) You have to classify movie reviews by people on a website into some classification scenario

4) There is a political party change in the state / county/district / country you live in and several people have tweeted or passed comments in twitter/facebook page feedback which you would like to analyse through AI based algorithms

All the above scenarios call for need to apply AI techniques to build a classification system which can further be used to "predict" the sentiment as more users provide their views in such scenarios.Sentiment analysis through AI techniques has frequent application in several industries and there could be several more examples than the ones stated above

Methods /Algorithms for Sentiment Analysis

Generally the 2 most common approaches to carry out sentiment analysis include

  1. Logistic Regression
  2. Naive Bayes algorithm

Both of the above approaches are discussed in the section of this blog. It should at the same time be underscored that more than the algorithm used for "classifying a sentiment" due importance has to be given to pre-processing the data available before passing it on to the algorithm. The pre-processing of the data has been discussed in the section of this blog. Insight is provided into libraries in an ML framework to be used for such pre-processing of the data.

This article emphasizes on the logistic regression classifier to perform a sentiment analysis as is part of the first course on NLP specialization
