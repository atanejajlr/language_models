1. INTRODUCTION

In my earlier articles, I had discussed the procedures generally followed for the solution of dynamic equilibrium equations:

  • Mode superposition method : https://www.linkedin.com/pulse/revisiting-mode-superposition-analysis-aka-modal-ajay-taneja?trk=mp-reader-card

Whereas, the mode superposition method involved transforming the problem from the global coordinate system to another coordinate system wherein the structural matrices are diagonal (before carrying out numerical integration), “direct” integration means that prior to the numerical integration, no transformation of the equations into a different form is carried out (unlike the mode superposition method). The integration schemes in the direct integration method detailed were: the explicit central difference method and the implicit Newmark method.

In the mode superposition, one needs to pre-estimate the number of modes to be included in the analysis i.e. (precisely) the number of equilibrium equations that one would solve (after transforming the problem to the modal space from the physical space. Analogous (and very much related) to mode superposition - in direct integration methods, one would need to have a pre-estimate of the time step to be used for the numerical integration scheme.

Selection of an appropriate time step in direct integration is of much importance. On one hand, the time step should be small enough to obtain accuracy in the solution; but, on the other hand, time step must not be smaller than necessary because with such time step the solution will be more costly than actually required.

The aim of this article is to describe practical considerations involving the selection of time step in direct integration schemes: implicit Newmark scheme and the explicit central difference scheme.

In order to get clear theoretical understanding of the practical approach involving selection of an appropriate time step, a logical background of the necessary theory is described in this article through the following sections (listed in order that they occur in this article);

  • Relationship between mode superposition and direct integration schemes
  • The essence of mode superposition
  • Estimation of time step for direct integration schemes
  • Steps to be considered in order to arrive an appropriate time step for direct integration schemes.

 

2. Relationship between mode superposition and direct integration methods

 

In mode superposition, we obtain ‘n’ uncoupled equations by transforming the problem to a different coordinate system than the global coordinate system. Each equation is independent of the other (since the resultant structural matrices in the new co-ordinate system are diagonal) and each equation is solved using Duhamel integral or (most often) the explicit central difference method. Since, the periods of vibration T = 2pi/ ωare known, we can choose in the numerical integration of the equations (that are obtained following the transformation of the co-ordinate system) an appropriate time step that requires the desired degree of accuracy. That is; a different time step can be chosen for each uncoupled equation.

On the other hand, if all n equations in mode superposition are integrated using the time step Δ t, then, the mode superposition is completely equivalent to a direct integration analysis in which the same integration scheme and the same time step Δ t are used. In other words, the solution of the finite element system equilibrium equations will be the same in either case.

Essence of mode superposition

Before going further into this discussion, let us first re-establish (or establish!) the essence of mode superposition method in our minds...

The analysis of a structure using mode superposition is the sum of two contributions;

1. A dynamic response obtained by multiplying the static response by a dynamic load factor (which in fact is the particular solution of the governing differential equation), and

2. An additional response which we call transient response.

The essence of a mode superposition solution of a dynamic response is that frequently only a small fraction of the total number of decoupled equations needs to be considered in order to obtain a good approximate solution to the exact solution of the differential equations;

Most frequently, only the first p equilibrium equations of the total ‘n’ equilibrium equations (where ‘n’ corresponds to the total degrees of freedom of the structure);

need to be used.

That is; we need to include in the analysis the equations only for i=1,2,….,p <<n, in order to obtain a good approximate solution.

That means, we need to solve only for the lowest p eigen values and corresponding eigen vectors of the problem in the equation;

And we only superimpose (or sum in) the responses in the first ‘p’ modes.

The physical reason for neglecting the higher modes can be understood from the graph below which shows a plot of the dynamic load factor as a function of omega_cap/omega i.e. the ratio of forcing frequencies and natural frequency.

It should be noted that, in general;

“The structure considered, spatial distribution and the frequency content of the loading determine the number of modes to be included in the analysis “

That is; in general, we can state that if we have high frequency modes, it is unlikely that those frequencies would be contained in the loading and thus be able to drive / resonate these high frequency modes.

But to ascertain the above, one would need to carry out a Fourier analysis and thus determine the frequency content of the loading.

Since, in direct integration the same time step is used for each equation of the above form, delta_t will have to be selected based on the smallest period in the system. As an estimate, delta_t, thus required, will have to be about Tn/10- where Tn corresponds to the smallest time period contained in the system (‘n’ is the number of degrees of freedom and Tn corresponds to the time period of the nth mode)

However, as discussed in the section on the essence of mode superposition and relationship between mode superposition and direct integration, the dynamic response lies in some modes of vibration and that for this reason only some mode shapes are considered in mode superposition analysis.

Therefore, the finite element idealization has to be chosen in such a way, lowest p frequencies and mode shapes of the structure are predicted accurately where p is determined by distribution and frequency content of loading.

It can therefore be concluded that in many analyses, we are interested integrating only the first p equations of the n equations accurately. This means that we would be able to revise delta_t to be about Tp/10 i.e. Tp/Tn smaller than the first estimate. In practical analysis the ratio Tp/Tn can be very large, to the order of 1000m meaning that the analysis would be much more effective using delta_t = Tp/10.

4. Practical approach to the selection of time step for direct integration schemes

 

From the above, one could conclude, that in mode superposition we neglect the higher modes in the analysis mainly because these   modes have such high frequencies that there are no driving forces available to resonate with them. This means that they will contribute very little to the motion, and, hence, they are unimportant in the dynamic analysis. It should be made sure that those higher frequencies are not contained in the loading.

Similarly, in direct integration schemes the time step Δ t may be chosen based on the highest frequency of loading contained in the system (or) smallest time period in the system.

The following approach most often proves useful to select the time step Δ t:

a) Identify the frequencies significantly contained in the loading, using a Fourier analysis if necessary. These frequencies may change as a function of time. Let the highest frequency significantly contained in the loading be ωu

b) Choose a finite element mesh that can accurately represent the static response and accurately represents all frequencies up to about: ωco = 4ωu

c) Perform the direct integration analysis. The time step delta_t for this solution should be equal to about Tco/20, where Tco = 2*pi/ ωco. This time step may be smaller for stability considerations when explicit central difference method but in most cases seems appropriate for implicit Newmark’s method or Wilson’s method
