A very short insight: What is Machine Learning all about?

Created on 2019-02-17 06:53

Published on 2019-02-17 06:55


(With due thanks to the course on courseera)

Machine Learning is about utilising mathematical theories and computer to solve problems similar to tasks human beings normally do.

Think about what a property agent goes when he talks to his customer when he ask "How much you think the potential prise rise of this house if I buy it?", or "What do you think this particular apartment would cost if I buy it with mortgage?"

These kinds of question must be explicitly programmed before we use Machine Learning. Which means programmer needs to know what precise list of questions customer may ask a property agent and create algorithms for each question, program it with some links to SQL database, in order for the program to do its work.

But with Machine Learning, no explicit programming is needed, first Data Scientist goes through table and tables of data to find insights (with the help of AI of course) for all possible interesting questions that can generates sales for this realty business.

Then, he can create a mathematical model with say, logistic regression, to make the model to learn a function for a certain task (aka answer questions), and put it live.

There is no need for specific algorithm and SQL program to create for each question, there is no need to program how to use each piece of SQL record, and there is no need to create a user interface for it, because NLP (Natural Language Processing) can do much of the work today with ML and Deep Learning.
